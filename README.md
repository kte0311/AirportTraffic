# AirportTraffic
날씨와 환율에 따른 인천국제공항의 이용객 수 변화 분석

1. Background
주제 : ‘날씨와 환율에 따른 인천국제공항의 이용객 수 변화 분석’ 여행으로 인천공항을 이용할 때면 이용객 수가 너무 많아 불편함을 겪곤 합니다.
이러한 이용객 수에 영향을 미치는 원인이 무엇일지 궁금하여 분석을 진행하게 되었습니다. 해당 분석은 인천국제공항의 이용객 수 변화 패턴을 파악함으로써 혼잡도를 줄이는 데에 기초자료가 될 것입니다.

3. Analysis 사용한 데이터
- y값 (2014~2023 인천국제공항 월별 승객 수, 출발/도착/종합)
- X값 (2014~2023 항공 기온, 강수량, 주요 국가 환율(미국, 중국, 일본))
